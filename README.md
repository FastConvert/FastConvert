# FastConvert

⚡ Free & secure file conversion tools.  
🛡️ Client-side only — your files never leave your device.  

## Available Tools
- JPG → PDF
- Word → PDF
- PDF Tools (merge, split, rotate, compress)
- Video → MP3 (soon)
